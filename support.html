2. 문의 가능한 항목

  • 계정 생성, 로그인, 인증 관련 문의
  • 거래소 연결 및 연결 해제 관련 문의
  • API Key 관리, 권한 설정, 보안 안내 관련 문의
  • 자산, 잔고, 주문, 체결, 거래 내역 표시 오류
  • 계정 삭제 또는 회원 탈퇴 요청
  • 개인정보 열람, 정정, 삭제, 처리 정지 요청
  • 앱 장애, 오류, 충돌, 접속 문제 신고

3. 보안상 보내면 안 되는 정보

이메일 문의 시 Secret Key 전체값, Access Token 전체값, 비밀번호, OTP/2FA 코드 전체를 보내지 마세요. 필요한 경우 일부 마스킹된 값과 상황 설명만 제공해 주세요.

  • 보내지 말아야 할 정보: 비밀번호, 전체 Secret Key, 전체 Access Token, OTP/2FA 코드, 복구 코드
  • 제공 가능한 정보 예시: 계정 이메일, 사용 기기, 앱 버전, 거래소명, 오류 발생 시각, 마스킹된 API Key 일부
  • 의심스러운 접근이 발생하면 거래소에서 API Key를 즉시 폐기하고 비밀번호를 변경하세요.

4. 문의 시 포함하면 좋은 정보

  • 사용 중인 기기와 운영체제 버전
  • Cryptory 앱 버전
  • 로그인 방식: [LOGIN_PROVIDERS]
  • 문제가 발생한 거래소명
  • 오류가 발생한 화면과 발생 시각
  • 문제를 재현할 수 있는 순서
  • 필요한 경우 스크린샷. 단, 민감 정보는 가려서 보내야 합니다.

5. 답변 소요 안내

일반 문의에 대한 응답 목표 기간은 [RESPONSE_TIME_TARGET]입니다. 장애, 보안, 거래소 API 관련 문의는 추가 확인이 필요할 수 있습니다.

6. 긴급 상황 안내

실제 거래, 출금, 체결, 계정 제한, 거래소 계정 접근 문제는 해당 거래소 고객센터에도 직접 문의해야 합니다. Cryptory는 거래소 자체가 아니며, 거래소 계정 조치나 체결 결과를 직접 변경할 수 없습니다.

  • 거래소 계정 도용이 의심되면 거래소에서 API Key 폐기, 비밀번호 변경, 2FA 재설정을 먼저 진행하세요.
  • 출금, 입금, 체결, 수수료, 계정 제한 문제는 해당 거래소 정책과 고객지원 절차를 따릅니다.
  • Cryptory 앱 표시 오류는 고객지원 이메일로 함께 알려 주세요.
