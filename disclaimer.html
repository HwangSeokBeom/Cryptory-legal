투자 유의 및 면책 안내

Cryptory의 시세, 비교, 시장 정보, 거래소 API 연동, 주문 및 잔고 표시와 관련한 유의사항을 안내합니다.

1. 투자 권유 아님

Cryptory가 제공하는 정보는 투자 권유, 투자 자문, 매수 또는 매도 추천이 아닙니다. 서비스 제공자는 투자 수익을 보장하지 않습니다.

이용자는 자신의 판단과 책임으로 가상자산 거래 여부를 결정해야 합니다.

2. 커뮤니티 게시글은 투자 조언이 아님

Cryptory 커뮤니티의 게시글, 댓글, 프로필, 링크 등 사용자 생성 콘텐츠는 작성자 개인의 의견입니다. 해당 콘텐츠는 투자 조언, 매수 또는 매도 추천, 거래 신호, 수익 보장이 아니며 Cryptory가 정확성이나 투자 결과를 보장하지 않습니다.

커뮤니티 이용 기준은 커뮤니티 운영 정책을 확인해 주세요.

3. 시세/비교/시장 정보는 참고용

모든 시세, 거래소별 비교, 시장 정보, 차트, 지표, 알림 또는 분석 정보는 참고용입니다. 정보는 지연, 오류, 누락, 거래소별 산정 방식 차이, 네트워크 상태에 영향을 받을 수 있습니다.

  • 거래 전 반드시 해당 거래소의 실제 주문 화면과 가격을 확인해야 합니다.
  • 거래소별 가격, 수수료, 호가, 유동성은 빠르게 변동될 수 있습니다.
  • 표시 정보와 실제 체결 결과가 다를 수 있습니다.

4. 거래소 API, 네트워크, 지연, 오류 가능성

Cryptory는 외부 거래소 API와 네트워크 상태에 의존할 수 있습니다. 거래소 점검, API 제한, 인증 오류, 네트워크 지연, 서버 장애, 앱 오류로 인해 정보 조회 또는 주문 요청이 실패하거나 지연될 수 있습니다.

  • 거래소 API 정책 변경으로 일부 기능이 제한될 수 있습니다.
  • 네트워크 또는 시스템 문제로 주문 요청 상태가 즉시 반영되지 않을 수 있습니다.
  • 장애 상황에서는 해당 거래소의 공식 화면과 공지를 우선 확인해야 합니다.

5. 주문/체결/잔고 정보의 최종 기준은 거래소

실제 주문, 체결, 잔고, 입출금, 수수료, 계정 제한, 거래 가능 여부의 최종 기준은 해당 거래소의 데이터와 정책입니다. Cryptory에 표시되는 정보는 거래소 API 응답을 기반으로 할 수 있으며, 최종 법적 또는 회계적 기준이 아닐 수 있습니다.

6. 손익 및 투자 판단 책임

가상자산 투자는 높은 변동성과 원금 손실 위험이 있습니다. 투자 판단, 주문 입력, 거래 결과, 손익, 세금, 법적 의무는 이용자 본인의 책임입니다.

이용자는 본인의 재정 상황과 위험 감수 능력을 고려하여 독립적으로 판단해야 합니다.

7. API 권한 및 보안 책임

거래소 API Key, Secret, Access Token, 비밀번호, OTP/2FA 코드, 거래소 계정 보안은 이용자가 안전하게 관리해야 합니다. Cryptory는 가능한 최소 권한 사용을 권장하며, 출금 권한이 포함된 API Key 사용은 권장하지 않습니다.

거래소 계정 이상 접근, API Key 유출, 의심스러운 주문이 발생하면 즉시 거래소에서 API Key를 폐기하고 해당 거래소 고객센터에 문의하세요.

8. 세금/법령 준수 책임

가상자산 거래와 관련한 세금 신고, 회계 처리, 법령 준수, 거래소 이용 자격 확인은 이용자 책임입니다. Cryptory는 세무, 법률, 투자 자문을 제공하지 않습니다.

준거법과 관할 관련 placeholder는 [GOVERNING_LAW] 및 [JURISDICTION]입니다.
