Delete Account

계정 삭제 안내

Cryptory 계정 삭제, 회원 탈퇴, 거래소 연결 해제, API Key 및 Token 폐기 방법을 안내합니다.

1. 계정 삭제 개요

이용자는 Cryptory 계정 삭제를 요청할 수 있습니다. 계정 삭제가 완료되면 서비스 이용을 위한 계정 정보와 연결 정보가 삭제 대상이 됩니다. 단, 법령상 보관 의무, 보안, 부정 이용 방지, 분쟁 대응을 위해 일부 정보가 제한된 기간 보관될 수 있습니다.

앱 내 삭제 경로가 확정되지 않은 경우에도 아래 placeholder를 실제 앱 화면 기준으로 교체해야 합니다.

2. 앱 내 탈퇴 방법

앱 내 계정 삭제 경로는 다음과 같습니다.

  1. Cryptory 앱을 엽니다.
  2. [APP_DELETE_ACCOUNT_PATH]로 이동합니다.
  3. 계정 삭제 안내를 확인합니다.
  4. 본인 확인 또는 확인 절차를 완료합니다.
  5. 계정 삭제 요청을 제출합니다.

3. 이메일 요청 방법

앱 내 계정 삭제가 어렵거나 접근할 수 없는 경우 이메일로 계정 삭제를 요청할 수 있습니다.

  • 요청 이메일: [SUPPORT_EMAIL]
  • 메일 제목 예시: Cryptory 계정 삭제 요청
  • 본문에 포함할 정보: 가입 이메일, 로그인 방식, 계정 식별에 필요한 최소 정보
  • 보안상 비밀번호, Secret Key 전체값, Access Token 전체값, OTP/2FA 코드는 보내지 마세요.

4. 삭제되는 정보

계정 삭제 완료 시 아래 정보가 삭제 대상이 됩니다.

  • Cryptory 계정 정보
  • 로그인 식별 정보
  • 앱 내 설정 및 연결 정보
  • Cryptory에 저장된 거래소 API Key, Secret, Access Token 등 연결 정보
  • 서비스 제공을 위해 저장된 자산 조회, 주문 조회, 거래 내역 관련 캐시 또는 기록

5. 일정 기간 보관될 수 있는 정보

다음 정보는 법령, 보안, 장애 대응, 부정 이용 방지, 분쟁 대응을 위해 일정 기간 보관될 수 있습니다.

  • 고객지원 문의 및 처리 이력
  • 접속 로그, 오류 로그, 보안 로그
  • 약관 위반 또는 부정 이용 대응에 필요한 최소 정보
  • 관련 법령상 보관이 필요한 정보

구체적인 보관 기간은 [DATA_RETENTION_PERIOD]입니다.

6. 거래소 API Key/Token 삭제 및 거래소 측 권한 폐기 안내

Cryptory 계정 삭제 또는 앱 삭제와 별개로, 거래소에서 생성한 API Key 또는 Token은 거래소 화면에서 직접 폐기해야 할 수 있습니다. 거래소 계정 보안을 위해 계정 삭제 전후로 거래소 API 관리 화면을 확인해 주세요.

  • 거래소 웹사이트 또는 앱의 API 관리 화면으로 이동합니다.
  • Cryptory 연결에 사용한 API Key를 찾습니다.
  • 해당 API Key를 삭제하거나 권한을 폐기합니다.
  • 출금 권한이 부여된 Key를 사용했다면 즉시 폐기하고 거래소 보안 설정을 점검합니다.

7. 처리 기간

계정 삭제 요청 처리 기간은 [ACCOUNT_DELETION_PROCESSING_TIME]입니다. 본인 확인이 필요하거나 요청 정보가 부족한 경우 추가 확인이 필요할 수 있습니다.
