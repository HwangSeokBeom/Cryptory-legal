Cryptory는 가상자산 시세 조회, 거래소별 비교, 자산 조회, 주문, 거래소 API 연동 기능을 제공하는 서비스입니다. Cryptory는 거래소 자체가 아니며, 이용자가 연결한 외부 거래소와 연동하여 정보를 조회하거나 이용자의 지시에 따라 요청을 전달할 수 있습니다.

서비스 개요

Cryptory는 가상자산 관련 정보를 확인하고 외부 거래소 계정을 연동해 일부 기능을 사용할 수 있도록 돕는 앱입니다.

시세 조회와 비교

가상자산 시세, 거래소별 가격 차이, 시장 정보를 확인할 수 있습니다. 표시 정보는 참고용일 수 있습니다.

자산 및 주문 확인

이용자가 연결한 거래소 API를 통해 잔고, 주문, 체결, 거래 내역 등 거래소 제공 정보를 조회할 수 있습니다.

외부 거래소 연동

Cryptory는 거래소가 아닙니다. 실제 계정, 주문, 체결, 출금, 수수료, 제한 사항은 해당 거래소 정책을 따릅니다.

보안 및 거래소 연동 안내

거래소 API Key, Secret, Access Token 등은 민감한 정보입니다. Cryptory는 필요한 범위의 권한만 사용하는 것을 권장하며, 출금 권한이 포함된 API Key 사용은 권장하지 않습니다. 실제 보관 방식과 보호 조치는 [API_STORAGE_DESCRIPTION]에 따라 운영됩니다.

투자 유의 요약

Cryptory가 제공하는 시세, 비교, 시장 정보는 투자 권유가 아닙니다. 가상자산 투자는 원금 손실 위험이 있으며, 투자 판단과 손익, 세금, 법적 의무, 거래소 계정 보안은 이용자 본인의 책임입니다.
