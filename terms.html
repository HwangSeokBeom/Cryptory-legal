이용약관

본 약관은 Cryptory 앱의 이용 조건, 거래소 연동, 주문 기능, 이용자 책임, 서비스 제공자의 책임 제한을 안내합니다.

1. 목적

본 약관은 [LEGAL_ENTITY_NAME]이 제공하는 Cryptory 서비스의 이용 조건과 절차, 이용자와 서비스 제공자의 권리, 의무, 책임 사항을 정하는 것을 목적으로 합니다.

2. 서비스 설명

Cryptory는 가상자산 시세 조회, 거래소별 비교, 자산 조회, 주문, 거래소 API 연동 기능을 제공하는 서비스입니다. Cryptory는 거래소 자체가 아니며, 가상자산 매매를 직접 중개하거나 보관하지 않습니다.

  • 가상자산 시세 및 시장 정보 조회
  • 거래소별 가격, 잔고, 주문, 체결, 거래 내역 표시
  • 이용자가 연결한 외부 거래소 API를 통한 정보 조회
  • 이용자의 지시에 따른 주문 요청 전달
  • 지원 거래소 목록: [SUPPORTED_EXCHANGES]

3. 계정 및 회원가입

이용자는 서비스 이용을 위해 회원가입 또는 로그인을 해야 할 수 있습니다. 로그인 제공자는 [LOGIN_PROVIDERS]입니다.

  • 이용자는 정확하고 최신의 계정 정보를 제공해야 합니다.
  • 계정 접근 권한, 비밀번호, 인증 수단, 기기 보안은 이용자가 관리해야 합니다.
  • 계정 도용 또는 무단 사용이 의심되면 즉시 고객지원으로 알려야 합니다.

4. 거래소 연동 및 API 사용

이용자는 외부 거래소에서 발급받은 API Key, Secret, Access Token 등을 입력하여 Cryptory와 거래소 계정을 연결할 수 있습니다. API 권한 설정과 거래소 계정 보안은 이용자의 책임입니다.

Cryptory는 가능한 최소 권한 사용을 권장합니다. 출금 권한이 포함된 API Key 사용은 권장하지 않습니다. 권장 또는 허용 권한 범위는 [API_PERMISSION_SCOPE]입니다.

  • 거래소 API 정보의 저장 및 보호 방식은 [API_STORAGE_DESCRIPTION]입니다.
  • 이용자는 거래소에서 직접 API Key를 폐기하거나 권한을 변경할 수 있습니다.
  • 거래소 API 정책, 사용 제한, 점검, 장애에 따라 서비스 기능이 제한될 수 있습니다.

5. 주문 기능 및 이용자 지시

Cryptory는 외부 거래소 API를 연결해 정보를 조회하거나, 이용자의 지시에 따라 주문 요청을 거래소에 전달할 수 있습니다. 주문의 최종 접수, 체결, 취소, 수수료, 제한, 실패 여부는 해당 거래소의 시스템과 정책에 따릅니다.

  • 주문 전 종목, 수량, 가격, 주문 유형, 거래소, 수수료를 이용자가 직접 확인해야 합니다.
  • 네트워크 지연, API 오류, 거래소 점검, 시장 급변으로 주문 결과가 예상과 다를 수 있습니다.
  • Cryptory는 주문 결과, 체결 가격, 체결 가능성, 투자 성과를 보장하지 않습니다.

6. 이용자 의무

  • 관련 법령, 본 약관, 거래소 약관, 앱 내 안내를 준수해야 합니다.
  • 본인의 투자 판단과 거래 결과를 직접 확인해야 합니다.
  • 거래소 계정, API Key, 인증 수단, 기기 보안을 안전하게 관리해야 합니다.
  • 타인의 계정 또는 API 정보를 무단으로 사용해서는 안 됩니다.

7. 금지행위

  • 타인의 계정, 거래소 API, 인증 정보를 무단으로 사용하는 행위
  • 서비스를 해킹, 우회, 변조하거나 비정상적인 자동화로 과도하게 호출하는 행위
  • 허위 정보 입력, 부정 거래, 시장 조작, 법령 위반 목적의 이용
  • 서비스 운영을 방해하거나 보안상 위험을 유발하는 행위
  • Cryptory를 투자 수익 보장 또는 공식 거래소 서비스로 오인하게 하는 행위

8. 커뮤니티 및 사용자 생성 콘텐츠

이용자가 커뮤니티에 게시하는 글, 댓글, 프로필명 등 사용자 생성 콘텐츠는 작성자 본인의 책임입니다. Cryptory는 커뮤니티 콘텐츠의 정확성, 투자 성과, 수익 가능성을 보장하지 않습니다.

  • 커뮤니티 게시글은 투자 조언, 매수 또는 매도 추천, 거래 신호가 아닙니다.
  • 이용자는 앱 내 기능을 통해 게시글, 댓글, 사용자를 신고하거나 다른 사용자를 차단할 수 있습니다.
  • 운영자는 정책 위반 콘텐츠를 숨김, 삭제, 제한할 수 있으며, 반복 위반 사용자의 커뮤니티 이용을 제한할 수 있습니다.
  • 자세한 기준은 커뮤니티 운영 정책을 따릅니다.

9. 서비스 변경, 제한, 중단

서비스 제공자는 운영, 보안, 법령, 거래소 정책, 기술적 사유에 따라 서비스의 전부 또는 일부를 변경, 제한, 중단할 수 있습니다.

  • 거래소 API 변경 또는 장애가 발생한 경우
  • 보안 사고, 점검, 시스템 장애가 발생한 경우
  • 법령 또는 앱 심사 기준 준수를 위해 필요한 경우
  • 서비스 운영상 기능 조정이 필요한 경우

10. 투자 판단 및 위험 부담

Cryptory는 투자 권유, 투자 자문, 수익 보장 서비스를 제공하지 않습니다. 가상자산 투자는 원금 손실 위험이 있으며, 투자 판단과 손익 책임은 이용자 본인에게 있습니다.

이용자는 자신의 재정 상황, 투자 목적, 위험 감수 능력을 고려하여 독립적으로 판단해야 합니다.

11. 책임 제한

서비스 제공자는 관련 법령에서 허용하는 범위 내에서 다음 사항에 대해 책임을 제한할 수 있습니다.

  • 외부 거래소 정책, 장애, 지연, API 오류, 점검으로 인한 손실
  • 이용자의 API 권한 설정, 계정 관리, 인증 정보 관리 부주의로 인한 손실
  • 시세 정보, 비교 정보, 시장 정보의 지연, 오류, 누락
  • 이용자의 투자 판단, 주문 입력, 거래 결과, 세금 또는 법적 의무

12. 계약 해지 및 계정 삭제

이용자는 언제든지 계정 삭제를 요청할 수 있습니다. 계정 삭제 절차는 계정 삭제 안내를 따릅니다.

앱 내 계정 삭제 경로는 [APP_DELETE_ACCOUNT_PATH]이며, 처리 기간은 [ACCOUNT_DELETION_PROCESSING_TIME]입니다.
